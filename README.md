# Banco de Dados - Cadastro de Estudante

Este repositório reúne uma atividade desenvolvida durante aula, com o objetivo de praticar a estrutura de um Banco de Dados e o desenvolvimento do mesmo utilizando o MySql Workbench.

## Conteúdo
Projeto desenvolvido em MySQL para simular um sistema de gestão acadêmica.

## Funcionalidades
- Cadastro de universidades
- Cadastro de faculdades
- Cadastro de cursos
- Cadastro de horários
- Cadastro de estudantes
- Cadastro de telefones
- Controle de matrículas

## Tecnologias
- MySQL

## Conceitos aplicados
- Modelagem relacional
- Chaves primárias e estrangeiras
- Integridade referencial
- DDL (CREATE, ALTER)
- DML (INSERT, UPDATE, DELETE)

## Estrutura do banco
Universidade → Faculdade → Curso → Estudante

Além disso, os cursos possuem horários e os estudantes possuem telefone e cadastro.
